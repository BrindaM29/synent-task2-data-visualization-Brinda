# Data Visualization using Iris Dataset

## Objective
Visualize patterns in the Iris dataset using Python.

## Dataset
Iris Dataset (Kaggle)

## Libraries Used
- Pandas
- Matplotlib
- Seaborn

## Visualizations
1. Bar Chart
2. Histogram
3. Scatter Plot (Petal Length vs Petal Width)
4. Scatter Plot (Sepal Length vs Sepal Width)
5. Correlation Heatmap

## How to Run

Install dependencies:

```bash
pip install -r requirements.txt
```

Run:

```bash
python iris_visualization.py
```

## Output
The program displays five visualizations that help understand the distribution and relationships between Iris flower features.
